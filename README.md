# MT5 止盈止损管理 GUI（Python）

一个基于 `MetaTrader5` 官方 Python 包 + `tkinter` 的简易桌面工具，用于批量管理持仓的止盈（TP）和止损（SL）。

## 功能

- 连接/断开 MT5 终端（支持填写 path/login/password/server）
- 拉取并展示当前持仓
- TP 支持两种模式切换：**按价格** / **按金额**
- SL 支持两种止损类型切换：
  - **普通止损**：按价格 / 按金额（二选一）
  - **固定移动止损**：按点数输入（每次更新按当前价重新计算）
- 可更新 **选中持仓** 或 **全部持仓**

## 安装

```bash
python -m pip install MetaTrader5
```

## 运行

```bash
python mt5_sl_tp_manager_gui.py
```

## 额外：MQL5 脚本版（可直接在 MT5 中运行）

仓库新增了 `MT5_SLTP_Manager.mq5`，用于在 MT5 内直接按输入参数批量更新持仓 SL/TP，逻辑与 Python GUI 版一致：

- TP：按价格 / 按金额（二选一，必填）
- SL：普通止损（按价格 / 按金额）或固定移动止损（点数）
- 支持按当前图表品种、指定 ticket、Magic 过滤持仓

使用方式（Windows MT5）：

1. 打开 MT5 数据文件夹，复制 `MT5_SLTP_Manager.mq5` 到 `MQL5/Scripts/`。
2. 在 MetaEditor 编译该脚本。
3. 在 MT5 导航器中运行脚本，按输入参数选择模式并执行。

## 使用说明

1. 打开 MT5 客户端并确保已登录交易账户。
2. 在 GUI 内点击“连接”。
3. 点击“刷新持仓”确认已加载持仓。
4. 在“止盈止损设置”填写参数：
   - **TP（必选一项）**
     - 按价格：直接输入 TP 目标价格
     - 按金额：输入目标盈利金额，程序自动换算 TP 价格
   - **SL**
     - 普通止损：按价格 / 按金额（二选一，必填）
     - 固定移动止损：输入点数（必填）
5. 点击“更新选中持仓”或“更新全部持仓”。

## 说明

- 当前版本按照“必填规则”执行：TP 必填，SL 在所选模式下必填。
- 不同品种最小止损距离、冻结级别等由经纪商服务器控制，若设置非法会被拒单并提示 `retcode`。
- “按金额”换算依赖 MT5 `order_calc_profit` 与品种参数；若经纪商未完整提供品种参数，可能导致换算失败并弹窗提示。
- 本工具仅示例用途，请先在模拟盘验证。
